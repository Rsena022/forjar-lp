# Forjar LP

Landing page comercial independente do sistema Forjar, preparada para publicação em um projeto separado na Vercel.

## Conectar o checkout

Abra `script.js` e cole o endereço da Cakto na constante `CHECKOUT_URL`.

```js
const CHECKOUT_URL = "https://pay.cakto.com.br/seu-checkout";
```

Todos os botões de compra da página usarão esse único endereço.
